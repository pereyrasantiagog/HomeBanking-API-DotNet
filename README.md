# 🏦 Home Banking API

Este proyecto es una API RESTful que simula las operaciones principales de un sistema bancario (Home Banking). Fue desarrollado como proyecto de portafolio para demostrar buenas prácticas en el diseño de bases de datos relacionales, manejo de transacciones y desarrollo Back End.

## 🚀 Tecnologías Utilizadas
* **Lenguaje:** C#
* **Framework:** ASP.NET Core Web API (.NET 10)
* **ORM:** Entity Framework Core (Enfoque Code-First)
* **Base de Datos:** MySQL
* **Documentación:** Swagger / OpenAPI

## ⚙️ Funcionalidades Actuales
* **Gestión de Usuarios:** Registro y consulta de clientes.
* **Gestión de Cuentas:** Asignación de cajas de ahorro con CBU y saldo a los usuarios.

## 🔜 Próximos Pasos
* Implementación del motor de transacciones (Transferencias, Depósitos y Retiros).
* Seguridad y Autenticación con JWT (JSON Web Tokens).
